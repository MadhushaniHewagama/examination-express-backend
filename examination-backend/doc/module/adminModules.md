# Show list of Modules where user is an admin

Get list of all modules where user is an admin

**URL** : `/modules/adminModules`

**Method** : `GET`

**Auth required** : No

**Permissions required** : None

**Parameters** :userId

## Success Response

**Code** : `200 OK`

**Content examples**


```json
{
    "success": true,
    "msg": [
        "cs2001"
    ]
}
```
